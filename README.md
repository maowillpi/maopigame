## 📱 使用教程

### 1. 打开游戏
在支持Web3的钱包浏览器中访问：
```
https://maowillpi.github.io/maopigame/game.html
```

### 2. 连接钱包
- 点击 "🔗 连接钱包开始游戏" 按钮
- 选择您的钱包（TP钱包、Trust Wallet等）
- 允许连接请求

### 3. 切换网络
系统会自动引导您切换到AlveyChain网络：
- **网络名称**：AlveyChain Mainnet
- **Chain ID**：3797 (0xED5)
- **RPC URL**：https://elves-core1.alvey.io
- **浏览器**：https://alveyscan.com

### 4. 选择代币
- 点击 **MAO** 或 **PI** 按钮选择游戏代币
- 查看您的代币余额

### 5. 授权代币
- 首次游戏需要授权代币使用权限
- 点击 "🔓 授权 MAO/PI 代币" 按钮
- 在钱包中确认授权交易

### 6. 开始游戏
- 确保余额充足（MAO≥100 或 PI≥10,000）
- 点击转盘中心的 "🎲 开始游戏" 按钮
- 等待转盘旋转和结果

### 7. 查看结果
- 转盘会自动停在对应的奖励位置
- 系统会弹出奖励结果
- 余额会自动更新
